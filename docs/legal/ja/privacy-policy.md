# ArcheryNow プライバシーポリシー

**最終更新日：2026年5月1日**

ArcheryNow（以下「本アプリ」）は、ユーザーのプライバシーを尊重します。本プライバシーポリシー（以下「本ポリシー」）では、本アプリがどのような情報を扱うか、どのように利用・共有するかを説明します。

> 本書は実装に沿ったドラフトであり、**法務上の助言ではありません。**

本ポリシー本文は [GitHub Pages](https://kitokito.app/privacy.html) で公開しています。[利用規約](https://kitokito.app/terms.html) もあわせてご確認ください。

---

## 1. 適用範囲

本ポリシーは、本アプリ（iOS）の利用において、当方が関与するデータの取り扱いに適用されます。有料購入に関する規約は [利用規約](terms-of-service.md) をご参照ください。

---

## 2. 当方が収集・利用する情報

### 2.1 アプリ内で主にローカルに保存されるデータ

次のデータは、**お使いのデバイス内（および、Pro で iCloud を利用する場合は Apple の iCloud 上のユーザーコンテナ）**に保存されます。当方が自前のサーバーにこれらを集約して保存する設定にはしていません。

- スコア、エンド、矢の記録、ターゲット上のプロット位置に関するデータ
- セッション、テンプレート、目標（ゴール）などの練習記録
- アプリ設定（表示モード、外観、言語、弓種のデフォルトなど）

アプリのアンインストールやデータ削除により、端末内のデータは失われる場合があります。iCloud の挙動は Apple の仕様に従います。

### 2.2 サードパーティサービスを通じて扱われるデータ

本アプリは、次の第三者サービスを利用しています。各社のプライバシーポリシーもあわせてご確認ください。

#### （1）課金・サブスクリプション（RevenueCat / Apple）

- **目的**: Pro の購入処理、購入状態の確認、復元
- **扱いが想定されるデータ**: 購入・サブスクリプションに関する情報、匿名化されたアプリ利用者 ID、デバイス・OS に関する技術情報など
- **参考**: [RevenueCat Privacy Policy](https://www.revenuecat.com/privacy)  
  課金そのものは **Apple** の決済により処理され、Apple のプライバシーポリシーも適用されます。

#### （2）iCloud（Pro の同期・バックアップ）

- **目的**: Pro 機能として、データベースやバックアップ JSON の同期・保存
- **扱い**: Apple の iCloud に保存されます。Apple の利用規約・プライバシーポリシーに従います。

#### （3）お問い合わせ（GitHub Issues）

- **目的**: フィードバック、不具合報告、開示請求の受付など
- **扱い**: 投稿内容、GitHub アカウントに紐づく公開情報などが **GitHub, Inc.** により処理されます。[GitHub のプライバシーに関する声明](https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement) をご確認ください。

#### （4）端末上の AI（対応機種・OS の場合）

- **目的**: コーチ機能における要約・応答の生成
- **扱い**: **原則として端末内**で処理され、当方のサーバーに会話内容を送信する実装は行っていません。Apple の AI 機能のログ等は Apple のポリシーに従います。

#### （5）広告配信・同意管理（Google AdMob / Google User Messaging Platform）

- **目的**: 広告表示、広告配信の安全性確保、広告効果測定、同意状態に応じたパーソナライズ広告または非パーソナライズ広告の配信
- **扱いが想定されるデータ**: 広告 ID（IDFA、ユーザーが ATT で許可した場合）、端末情報、IP アドレス、広告の表示・クリック等の利用状況データ、同意状態に関する情報など
- **参考**: [Google Privacy Policy](https://policies.google.com/privacy)、[Google の広告に関する説明](https://policies.google.com/technologies/ads)
- **選択肢**: EEA、英国、スイス等では Google User Messaging Platform による同意メッセージを表示し、同意状態に応じて広告リクエストを制御します。iOS の ATT ダイアログで許可しない場合、IDFA を利用したトラッキングは行われません。アプリ内に「プライバシー設定」が表示される場合は、そこから広告同意の選択を見直せます。

### 2.3 トラッキング（広告 ID 等）

本アプリは Google AdMob を利用するため、ユーザーの同意および iOS の ATT 許可状況に応じて、広告 ID や利用状況データが広告配信・測定に利用される場合があります。パーソナライズ広告が許可されない場合でも、非パーソナライズ広告や SKAdNetwork 等による集計的な広告測定が行われる場合があります。

## 3. 利用目的

取得・処理する情報は、次の目的の範囲で利用します。

1. 本アプリの提供（記録、統計、表示、エクスポート、Pro 機能）
2. 課金状態の確認、不正利用の防止、サポート対応
3. 広告表示、広告配信の安全性確保、同意状態に応じた広告配信
4. 品質改善、不具合解析（個人を特定しない形の集計を含む）
5. 法令に基づく対応

目的外の利用はしません。

- **Issues の投稿**: GitHub 上に残る期間は GitHub のポリシーに従います。

## 5. データの共有・開示・国外への転送

当方は、次の場合を除き、**個人情報を第三者に販売したり、マーケティング目的で共有したりしません。**

- 法令に基づく開示請求、裁判所の命令に従う場合

- 本ポリシーまたは各サードパーティのポリシーに記載された範囲での、課金・ホスティング・広告配信等の処理の委託（Apple、RevenueCat、GitHub、Google 等）

**外国にある第三者への提供（個人情報保護法上の参考）**  
Apple、RevenueCat、GitHub、Google などは、**国外（主に米国等）にサーバー等を有する**場合があります。これらの事業者への提供にあたっては、各社のプライバシーポリシーに記載のとおり、契約・認定制度その他の措置が講じられることがあります。当該提供を望まない場合は、該当機能（課金、広告視聴による機能解放、Issues での問い合わせ等）をご利用いただけない場合があります。詳細は各社のポリシーをご確認ください。

## 8. 欧州経済領域（EEA）・英国のユーザー向け（GDPR 等）

本アプリを **EEA または英国**から利用される場合、適用法に基づき、次の権利を有する場合があります（要否・詳細は法域により異なります）。

- アクセス、訂正、消去、処理の制限、データポータビリティ、処理への異議申立て

**法的根拠の例**: サービス提供（契約の履行）、正当な利益（セキュリティ・改善）、同意（該当する場合）など。  
**越境移転**: 前記のとおり Apple・RevenueCat・GitHub・Google 等への提供が生じ得ます。各社のポリシーおよび標準契約条項等の枠組みが適用されることがあります。  
ご請求は GitHub Issues から可能な範囲で受け付けます。必要に応じて追加の本人確認をお願いすることがあります。

## 9. カリフォルニア州居住者向け（CCPA / CPRA）

カリフォルニア州の「消費者」に該当する場合、州法に基づき、次の権利等を有する場合があります。

- **知る権利**（収集・開示された個人情報のカテゴリー・目的等）
- **削除請求権**（一定の例外あり）
- **オプトアウト権**（「販売」または「共有」— 本アプリは**個人情報の販売を行っておらず**、行う予定もありません）
- **感応的な個人情報の制限**（該当する処理を行う場合の制限請求）
- **非差別待遇**

AdMob によるパーソナライズ広告がカリフォルニア州法上の「共有」に該当する場合があります。該当する場合は、同意管理画面、OS 設定、またはアプリ内のプライバシー設定から選択を変更できるようにします。ご請求は GitHub Issues からお願いします。代理人による請求には、法令に従い認証が必要な場合があります。
