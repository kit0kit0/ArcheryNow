## 2. 取得する情報の種類

### 2.1 ユーザーが本アプリに入力・生成する情報

本アプリの機能のために、端末内に次のような情報が保存・処理されることがあります（**主に端末ローカル**）。

- スコア、エンド、矢の記録、ターゲット上のプロット位置に関するデータ
- セッション、テンプレート、目標（ゴール）等の練習記録
- アプリ設定（表示モード、外観、言語の好設定、弓種のデフォルト等）

これらは**アーチェリーの記録・分析・表示**のために利用されます。

### 2.2 端末・アプリの動作に伴い自動的に扱われる情報

- **UserDefaults** 等による設定・フラグの保存（オンボーディング完了、プレミアム状態のキャッシュ等）
- データベースファイルやバックアップファイルの**タイムスタンプ**等、ファイルシステムが付与する情報

本アプリのプライバシーマニフェスト上、**アプリによるトラッキング（Tracking）は行っていません**（`NSPrivacyTracking`: false、収集データ型の宣言なし）。

### 2.3 課金・サブスクリプションに関連する情報

- **Apple Inc.**（App Store）を通じた購入・サブスクリプションの取引が発生します。Apple は購入履歴、Apple ID に紐づく課金情報等を保有します。
- 当社は、エンタイトルメント（Pro 権利）の確認等のため **RevenueCat, Inc.** の SDK を利用する場合があります。RevenueCat には、匿名化されたアプリ利用者 ID、購入イベント、端末・OS に関する技術情報等が送信されることがあります。詳細は [RevenueCat のプライバシー情報](https://www.revenuecat.com/privacy) をご確認ください。

### 2.4 iCloud（クラウド同期・バックアップ）

- **Pro（有料）機能**として、ユーザーの iCloud が有効な環境では、データベースやバックアップ JSON が **Apple の iCloud** 上のユーザーコンテナに保存・同期される設計があります。
- 当該データは **Apple の iCloud 利用規約およびプライバシーポリシー**に従って処理されます。

### 2.5 コーチ（AI）機能

- 対応する端末・OS では、**端末上の AI（Apple の Foundation Models 等）**により、スコアや統計を要約したコーチング文やチャット応答を生成する場合があります。この処理は**原則として端末内**で行われ、当社が独自のサーバーに会話内容を送信する実装は行っていません。  
- プラットフォーム提供者（Apple）の AI 機能の動作・ログは、Apple のポリシーに従います。

### 2.6 お問い合わせ（GitHub Issues 等）

- **GitHub Issues** 等を通じてお問い合わせいただいた場合、**投稿内容、GitHub 上のユーザー名・プロフィール等、やり取りの記録**が、当社（またはリポジトリ管理者）および **GitHub, Inc.** により、対応・不正利用防止・サービス運営のために処理・保存されることがあります。GitHub 上の取扱いは [GitHub のプライバシーに関する声明](https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement) をご確認ください。
- **App Store** 経由のお問い合わせについては、**Apple Inc.** のポリシーに従い処理されます。

### 2.7 個人情報に該当しない情報

- 上記のうち、単体では特定の個人を識別できない統計・技術情報は、個人情報保護法上の「個人情報」に該当しない場合があります。ただし、他の情報と結合して個人を識別できるようになった場合には個人情報として取り扱います。

## 3. 利用目的

取得した情報は、次の目的の範囲内で利用します。

1. 本アプリの提供、機能の実現（記録、統計、表示、エクスポート、Pro 機能）
2. 課金状態の確認、不正利用の防止、カスタマーサポート
3. 本アプリおよびサービスの改善、不具合解析（個人を特定しない形態の集計を含む）
4. 法令に基づく対応、権利保護

**目的外の利用は行いません。** 目的を変更する場合は、法令に従い必要な手続を行います。

## 5. 保存期間

- **端末内データ**: ユーザーがアプリを削除する、またはデータを消去するまで。iCloud 利用時は、Apple の仕様およびユーザーの iCloud 設定に従います。
- **お問い合わせ記録（GitHub Issues 等）**: GitHub 上に残る投稿は GitHub のポリシーに従います。当社が別途ローカルに保存する場合は、対応完了後、合理的な期間を経過した後に削除します（法令上の保存義務がある場合を除く）。
- **RevenueCat / Apple 上の課金関連データ**: 各事業者のポリシーに従います。
