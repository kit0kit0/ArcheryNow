# ArcheryNow（法務文書・GitHub Pages）

ArcheryNow アプリの**利用規約**と**プライバシーポリシー**を公開するリポジトリです。アプリ本体のソースは含みません。

## 公開 URL（GitHub Pages 有効化後）

リポジトリ **Settings → Pages** で **Deploy from a branch** を選び、Branch を **`main`**、Folder を **`/docs`** に設定してください。反映後、おおむね次の URL で閲覧できます。

| 文書 | URL |
|------|-----|
| トップ | https://kit0kit0.github.io/ArcheryNow/ |
| 利用規約 | https://kit0kit0.github.io/ArcheryNow/terms.html |
| プライバシー | https://kit0kit0.github.io/ArcheryNow/privacy.html |

## 構成（スキル `github-pages-legal-docs` に準拠）

- `docs/legal/terms-of-service.md` … 利用規約（Jekyll の `permalink` で `/terms.html`）
- `docs/legal/privacy-policy.md` … プライバシーポリシー（`/privacy.html`）
- `docs/_layouts/default.html` … 共通レイアウト
- `docs/_config.yml` … Jekyll 設定
