# ArcheryNow（法務文書・GitHub Pages）

ArcheryNow アプリの**利用規約**と**プライバシーポリシー**を多言語で公開するリポジトリです。アプリ本体のソースは含みません。

## GitHub Pages

**Settings → Pages** で Branch **`main`**、Folder **`/docs`** を選択してください。

### デフォルト URL（アプリ内 `LegalLinks` と一致）

| 文書 | URL |
|------|-----|
| サイトトップ（言語一覧） | https://kit0kit0.github.io/ArcheryNow/ |
| 利用規約（**日本語・既定**） | https://kit0kit0.github.io/ArcheryNow/terms.html |
| プライバシー（**日本語・既定**） | https://kit0kit0.github.io/ArcheryNow/privacy.html |

### app-ads.txt（Google AdMob）

| ファイル | 公開 URL |
|----------|----------|
| `docs/app-ads.txt` | https://kit0kit0.github.io/ArcheryNow/app-ads.txt |

ストアの開発者サイト URL が `https://kit0kit0.github.io/ArcheryNow/` の場合、[AdMob のクローラ](https://support.google.com/admob/answer/9363762)は **`https://kit0kit0.github.io/app-ads.txt`**（`github.io` のルート）も探索します。AdMob コンソールで「見つからない」となるときは、リポジトリ **`kit0kit0.github.io`**（ユーザーサイト）のルートに同内容の `app-ads.txt` を置く、またはカスタムドメインのサイトルートに配置するなど、[仕様の「ホスト名」](https://support.google.com/admob/answer/9363762)に合わせてください。
